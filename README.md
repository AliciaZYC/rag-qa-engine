# RAG QA Engine

## Tech Stack

**Frontend:** React + Vite  
**Backend:** Flask (Python)  
**Containerization:** Docker + Docker Compose

## How to Run

```bash
docker-compose up --build
```

- Frontend: http://localhost:5173
- Backend API: http://localhost:5000

**Note:** If localhost doesn't work, use http://127.0.0.1:5173

## Stop

```bash
docker-compose down
```
